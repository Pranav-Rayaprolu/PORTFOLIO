## ğŸ Getting Started

### 1. Clone the repository

```sh
git clone https://github.com/Pranav-Rayaprolu/LifeSync.ai.git
cd "ai-portfolio version 1"
```

### 2. Install dependencies

```sh
pnpm install
# or
npm install

```
